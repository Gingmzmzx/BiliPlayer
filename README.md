# BiliPlayer — B 站音乐播放器

自动读取 B 站收藏夹视频，以纯音频模式播放。支持 Web 远程控制、封面展示、播放偏好设置。

---

## ✨ 功能

### 桌面端
- 自动抓取指定 UID 收藏夹内的视频，提取音频播放
- 右侧垂直进度条（置顶、小巧、不干扰工作）
- 进度条悬浮显示当前播放信息（封面 + 标题 + BV 号）
- 进度条拖拽跳转 / 点击暂停
- 菜单栏托盘图标，单击打开控制面板
- 首次启动弹出配置向导（UID、收藏夹名、音量、端口），之后自动记住

### Web 控制面板（`http://localhost:58000`）
- 🎧 **播放器** — 封面大图、标题、进度条、播放/下一首、音量
- 📋 **播放列表** — 查看 / 点击切换 / 排序 / 删除 / 手动添加 BV 号
- 🔄 **播放模式** — 顺序 / 随机 / 单曲循环
- 👤 **用户切换** — 修改 UID 和收藏夹，即时重载
- ⭐ **播放偏好** — 按视频设置起始位置、结束位置、分 P（弹窗管理）
- 🎨 **主题色切换** — 粉 / 蓝 / 紫 / 绿 / 琥珀
- 💾 **配置持久化** — 用户、音量、偏好自动保存
- 📱 **响应式** — 适配桌面和手机浏览器

---

## 🚀 安装与运行

```bash
git clone <repo-url> && cd BiliPlayer
pip install -r requirements.txt
playwright install chromium
python main.py
```

首次运行弹出配置窗口，填写 B 站 UID 和收藏夹名称后开始播放。
